# Yao-Pos-System

這是一個飲料的點餐系統，它使用 HTML、 CSS、 JavaScript 和 Bootstrap 建構，可讓您點選飲料項目以及冰塊量和甜度，最後可看到各飲料項目明細並結算出總消費金額。

![pic2](https://imgur.com/d9KA843.png)

## Features

|Functions|Detail|
|:--- | --- |
| 流程 | 1. 點選右方飲料點餐區的品項<br>2. 選擇冰塊量<br>3. 選擇甜度<br>4. 新增飲料或刪除左方訂單區的訂單<br>5. 結帳 |
| 新增訂單 | 新增至左方訂單區可供查看飲料品項明細及費用 | 
| 假如沒點到飲料品項卻新增飲料 | 系統會發出提醒訊息：請選擇飲料項目 |
| 刪除訂單 | 刪除左方飲料訂單區的訂單卡 |
| 結帳 | 系統自動結算金額並跳出新視窗提示消費總金額 |
